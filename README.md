# Simp
Simple programming landuage that is inspired by C++ and Python but nothing like either one. I know C++ and Python quite well, but there are things that I don't like about both of them. There are things that I like about them, too. 

## Why?
This is a hobby project that I have been working on for a while. There is no real expectation that this will be useful for any particular purpose. I am going through all of the slog to make it happen because it's fun. 

## General Plan
Simp is intended to be a general purpose application language that combines the best features of C++ and Python. The general idea is to implement the language with the minimum effort required, but still have it able to do everything that it needs to do. I plan to translate the Simp language to C. 

## Features
- Namespaces. 
- Classes (single inheretance).
- Exceptions.
- Iterators.
- No switch/case or for clauses.
- Formatting strings is more or less automatic.
- No pointers. 
- Managed memory.
